hot_key (типо горячий ключ)

1 -> запуск камеры ЧЕРНАЯ

2 -> другой цветокор

3 -> POV камера 4K ULTRA HD 5000FPS (фронтальная камера)

: -> запуск камеры отладки 

\ -> Камера слежки за объектом (не является камерой бпла, камера среды GRID)

. -> возврат на камеру от третьего лица

l -> для переключения на первое лицо, фронтальная камера

Модель БПЛА:

Основная -> Freefly Alta X (текущая)

Под кастомку -> Holybro S500 (использовалась на ранних тестах)

Примечание!

Ввиду своей кривезны боксов модели по типу комбинированного бпла с двумя неподвижными крылами и 4 ротарами, а так же большая модель Мультикоптера, не использовалась в тестах. 


Локации:

Для теста большой дистанции -> Blocks (но сомнительно)

Для тестов камеры, лидара, Gps -> Nordic Forest

Для условных тестов работоспособности -> City Block (тест обхода зданий, город)


Sensor:

Resolution -> взять более хорошое разрешение камеры, Fow тот же, 1280 x 720

Подключить -> Lidar

Подключить -> Distance (вроде отрисовка, но Draw Debug Points отвечает за трасеры, так что хз)


Модули:

Navigation

-> Visual Servoing (ИИ камера)

-> Inspection (иследование объектов по пути)

Object Detection

-> GroundingDINO (детект объектов, камера)

-> RT-DETR (более шустрый детект)

Segmentation (понимание текста, локализация объектов)

-> Segment Anything 2

-> GroundedSAM

Monocular Depth (на всякий случай)

-> MiDaS (катра глубины)

-> ZoeDepth (альтернатива, карман не тянет)
